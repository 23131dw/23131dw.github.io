记录过程以及遇到的问题。

戴尔主机装Ubuntu

1.官网下载Ubuntu的iso文件放入u盘中。

2.电脑开机，按F12进入BIOS界面，介质从硬盘改为USB，选择启动方式以 “USB” 引导加载。

3.常规安装选择“Try or Install Ubuntu”‌，遇到显卡问题时选择“Ubuntu (safe graphics)”‌，通常都是常规安装。

4.经过上面的选择，就会进入ubuntu安装的一系列节点。

注意：

1.ubuntu的安装选项需要选择中文方便后续查看。

2.安装中文输入法，主要有IBus以及Fcitx，这里拿IBus为例：

(1)在setting中选择language and region ,然后“Manage Installed Languages”，“Install/Remove Languages”，“Chinese(Simplified)”，“Apply”

(2)执行如下命令：

sudo apt update

sudo apt-get install ibus ibus-clutter ibus-gtk ibus-gtk3 ibus-qt4

(3)终端运行ibus-setup，点击“input method”，添加“chineses”-“intelligent pinyin”，然后关闭。

(4)在“设置setting”-“language and region”-“input source”中，点击“+”、点击“chinese”，添加“chineses”-“intelligent pinyin””。此时可以看到已经有中文输入法。中英文切换方式win+space。

Ubuntu下使用virt-manager

1.首先安装virt-manager运行如下命令：

sudo apt-get install qemu qemu-kvm qemu-system -y

sudo apt-get install virt-manager -y

2.终端界面输入virt-manager并且选择创建新的虚拟机。

3.选择本地安装介质iso，并且准备好需要运行的iso文件。

4.根据iso安装的路径选择，这里需要注意的是下方的检测，有的iso文件可以自动检测，但是有的不行，如果不西行的话可以自己填入一个ubuntu版本。

5.选择cpu核数以及内存大小。

6.选择自定义的存储，最好大于50G。

7.最后可以选择自定义的安装配置。

拥有自己名字的博客

1.在github创建一个新的仓库，名字叫做：账户名.github.io

2.在创建好的仓库下创建index.html文件，然后就可以根据账户名.github.io来访问这个html。

3.注册阿里云，查找自己想要的域名是否被注册，例如自己的姓名。

4.进入阿里云，云解析DNS，权威域名解析进行如下设置。

(1)主机记录值：www 记录类型：CNAME 记录值：自己注册的域名

(2)主机记录值：@ 记录类型：A 记录值：185.199.108.153

5.进入github仓库设置Custom domain，设置成自己的域名。

6.阿里云一定要实名认证域名才能够使用。

7.最后可以根据自己的域名进行网站的访问。

编译Linux内核

1.在如下连接下载某个版本的Linux并且解压：

https://git.kernel.org/pub/scm/linux/kernel/git/torvalds/linux.git/

2.进入下载好的Linux内核代码运行如下命令进行一些配置：

make O=build menuconfig

3.编译内核镜像，会在/build下生成vmlinux,/build/arch/x86_64生成bzImage：

make O=build bzImage -j8

vmlinux 文件是通过编译和链接内核代码生成的原始 ELF 文件，未经过压缩，包含符号和重定位信息。

bzImage 则是从 vmlinux 派生而来的，经过压缩和处理的内核映像文件。
